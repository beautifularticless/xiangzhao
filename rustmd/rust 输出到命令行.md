rust 输出到命令行

## Rust 命令行输出基础

在 Rust 中， **最常用、最核心** 的命令行输出方式是 **宏** ： `println!` 和 `print!`，它们是标准库内置的，无需额外依赖。

## 1\. 最常用：换行输出 println!

`println!` 会 **自动在末尾添加换行符** ，直接打印内容并换行。

### 基础用法

```rust
fn main() {
    // 直接打印字符串
    println!("Hello, Rust!");

    // 打印变量（占位符 {}）
    let name = "小明";
    println!("你好，{}", name); // 输出：你好，小明

    // 打印多个变量
    let age = 18;
    println!("姓名：{}，年龄：{}", name, age);
}
```

## 2\. 不换行输出 print!

`print!` **不会自动换行** ，适合连续输出内容。

> 注意：标准输出默认有缓冲，需要用 `std::io::Write` 刷新缓冲区才能立即显示。

```rust
use std::io::{self, Write}; // 导入刷新依赖

fn main() {
    print!("加载中...");
    // 强制刷新缓冲区，让内容立即显示
    io::stdout().flush().unwrap();
    print!("完成！\n"); // 手动加换行
}
```

## 3\. 格式化输出（进阶）

Rust 支持强大的 **格式化语法** ，满足数字、对齐、精度等需求：

### 常用格式化场景

```rust
fn main() {
    // 1. 数字格式化
    let num = 123;
    println!("十进制：{}", num);    // 普通数字
    println!("十六进制：{:x}", num); // 小写十六进制
    println!("二进制：{:b}", num);   // 二进制

    // 2. 浮点数精度（保留2位小数）
    let pi = 3.1415926;
    println!("π ≈ {:.2}", pi); // 输出：3.14

    // 3. 对齐输出（右对齐，占10个字符宽度）
    println!("|{:>10}|", "Rust"); // 输出：|      Rust|

    // 4. 命名参数（可读性更强）
    println!("语言：{lang}，版本：{ver}", lang="Rust", ver="1.75");
}
```

## 4\. 调试输出 dbg!

开发调试时专用， **自带文件名、行号、变量值** ，比 `println!` 更适合调试：

```rust
fn main() {
    let score = 95;
    dbg!(score); 
    // 输出：[src/main.rs:4] score = 95
}
```

---

### 快速总结

1. **换行输出** ： `println!("内容")` ✅ 最常用
2. **不换行输出** ： `print!("内容")` + 手动刷新
3. **格式化** ：用 `{}` 占位， `{:.2}` 控制精度等
4. **调试** ：专用 `dbg!(变量)`

这些是 Rust 命令行输出的全部基础用法，覆盖 99% 的日常场景。
