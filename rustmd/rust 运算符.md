## Rust 运算符

Rust 运算符和其他语言（C/C++、Java）很像，但 **更安全、更严格** ，没有隐式类型转换，语法清晰。

我把 Rust 常用运算符分成 **6 大类** ，一次性给你讲全，方便直接对照使用。

---

## 1\. 算术运算符

用于数值计算（整数、浮点数）

表格

| 运算符 | 作用 | 示例 |
| --- | --- | --- |
| `+` | 加法 | `a + b` |
| `-` | 减法 | `a - b` |
| `*` | 乘法 | `a * b` |
| `/` | 除法（整数会截断） | `5 / 2 = 2` |
| `%` | 取余 | `5 % 2 = 1` |

⚠ **注意** ：

- Rust **不支持** `++` / `--`
- 整数除法会 **向下取整** ，不会自动变浮点数

```rust
let a = 5i32;
let b = 2i32;
println!("{}", a / b); // 输出 2
```

---

## 2\. 比较运算符

返回 **bool** 类型（ `true` / `false` ）

表格

| 运算符 | 作用 |
| --- | --- |
| `==` | 等于 |
| `!=` | 不等于 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

示例：

```rust
let x = 10;
println!("{}", x > 5); // true
```

---

## 3\. 逻辑运算符

用于布尔判断

表格

| 运算符 | 作用 |  |  |
| --- | --- | --- | --- |
| `&&` | 逻辑与（短路） |  |  |
| \` |  | \` | 逻辑或（短路） |
| `!` | 逻辑非 |  |  |

```rust
let a = true;
let b = false;
println!("{}", a && b); // false
```

---

## 4\. 位运算符

对二进制位操作（底层常用）

表格

| 运算符 | 作用 |  |
| --- | --- | --- |
| `&` | 按位与 |  |
| \` | \` | 按位或 |
| `^` | 按位异或 |  |
| `!` | 按位非（也写作 `~` ） |  |
| `<<` | 左移 |  |
| `>>` | 右移 |  |

```rust
let a = 0b1010;
let b = 0b1100;
println!("{:b}", a & b); // 1000
```

---

## 5\. 赋值与复合赋值

Rust 没有 `+=` 之外的花哨赋值，非常简洁

表格

| 运算符 | 作用 |  |
| --- | --- | --- |
| `=` | 赋值 |  |
| `+=` | 加后赋值 |  |
| `-=` | 减后赋值 |  |
| `*=` | 乘后赋值 |  |
| `/=` | 除后赋值 |  |
| `%=` | 取余后赋值 |  |
| `&=` | 位与后赋值 |  |
| \` | \=\` | 位或后赋值 |
| `^=` | 位异或后赋值 |  |
| `<<=` | 左移后赋值 |  |
| `>>=` | 右移后赋值 |  |

示例：

```rust
let mut x = 5;
x += 3; // x = 8
```

## 6\. 特殊运算符（Rust 独有 / 常用）

这些是 Rust 高频使用的特色运算符：

### （1）解引用与取地址

- `&` 取引用（借用）
- `*` 解引用

```rust
let x = 10;
let y = &x;
println!("{}", *y); // 10
```

### （2）类型转换 as

Rust **不允许隐式转换** ，必须用 `as`

```rust
let a = 5i32;
let b = a as f64;
```

### （3）范围运算符

- `..` 左闭右开（不包含结束）
- `..=` 全闭区间（包含结束）

```rust
1..5    // 1,2,3,4
1..=5   // 1,2,3,4,5
```

### （4）成员访问

- `.` 访问结构体 / 枚举成员
- `::` 访问命名空间 / 关联函数

```rust
String::new(); // 用 ::
"hello".len(); // 用 .
```

### （5）问号错误传播?

Rust 最标志性的错误处理运算符

```rust
fn f() -> Result<i32, Error> {
    let x = some_func()?;
    Ok(x)
}
```

---

## 运算符优先级（从高到低）

记不住就用 **括号** ，最安全！

1. 一元：`!` `*` `&` `-`
2. 乘除模： `*` `/` `%`
3. 加减： `+` `-`
4. 移位： `<<` `>>`
5. 比较： `==` `!=` `<` `>` `<=` `>=`
6. 逻辑： `&&` → `||`
7. 赋值： `=` `+=` 等

---

## 小总结

- Rust **没有 `++` / `--`**
- 没有隐式类型转换，必须用 `as`
- 逻辑运算符是短路的
- 范围运算符 `..` 和 `..=` 非常常用
- `?` 是 Rust 特色错误处理运算符

---

### 总结

1. Rust 运算符分为 **算术、比较、逻辑、位运算、赋值、特殊运算符** 六类
2. 核心特性： **无 `++/--` 、无隐式转换、严格安全**
3. 高频特色运算符： `&/*` （引用 / 解引用）、 `as` （类型转换）、`..`（范围）、`?`（错误传播）、`::`（命名空间访问）
4. 优先级记不住时， **括号是最安全的选择**
