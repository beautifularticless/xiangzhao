## Rust 函数基础

函数是 Rust 代码的核心组成部分，用于封装可复用的逻辑。Rust 函数语法简洁、安全，且严格遵循 **类型安全** 规则。

## 1\. 基本语法

Rust 使用 `fn` 关键字声明函数，格式：

```rust
// 函数定义
fn 函数名(参数1: 类型1, 参数2: 类型2) -> 返回值类型 {
    // 函数体
    // 最后一行表达式作为返回值（无需分号），或用 return 关键字
}
```

### 核心规则

- 函数名采用 **蛇形命名法** （ `snake_case` ）： `calculate_sum` 、 `print_info`
- 参数 **必须声明类型**
- 返回值用 `-> 类型` 声明，无返回值可省略（默认返回单元类型 `()` ）

## 2\. 简单示例

### 无参数、无返回值

```rust
// 定义函数
fn greet() {
    println!("Hello, Rust Function!");
}

fn main() {
    greet(); // 调用函数
}
```

### 带参数的函数

```rust
// 两个 i32 类型参数
fn add(a: i32, b: i32) {
    println!("{} + {} = {}", a, b, a + b);
}

fn main() {
    add(5, 3); // 输出：5 + 3 = 8
}
```

### 带返回值的函数

Rust 有两种返回方式：

1. **表达式返回** （推荐）：函数最后一行 **不带分号** 的表达式作为返回值
2. **return 关键字** ：可提前返回

```rust
// 计算两数之和，返回 i32 类型
fn sum(a: i32, b: i32) -> i32 {
    a + b // 无分号 = 返回值
}

// 带 return 的写法
fn is_positive(n: i32) -> bool {
    if n > 0 {
        return true; // 提前返回
    }
    false
}

fn main() {
    let result = sum(10, 20);
    println!("结果：{}", result); // 30
    println!("正数？{}", is_positive(-5)); // false
}
```

## 3\. 特殊：单元类型 ()

如果函数没有返回值，Rust 默认返回 **单元类型 `()`** （表示空值），可以省略不写：

```rust
// 等价于 fn print() -> ()
fn print() {
    println!("无返回值");
}
```

## 4\. 函数进阶

### 函数作为参数（高阶函数）

Rust 支持将函数作为参数传递，是函数式编程特性：

```rust
// 接收一个函数作为参数
fn execute<F: Fn()>(func: F) {
    func();
}

fn hello() {
    println!("高阶函数调用！");
}

fn main() {
    execute(hello);
}
```

### 方法（关联函数）

函数可以绑定到结构体 / 枚举上，称为 **方法** （用 `impl` 定义）：

```rust
struct Calculator;

impl Calculator {
    // 关联函数（类似静态方法）
    fn add(a: i32, b: i32) -> i32 {
        a + b
    }
}

fn main() {
    let res = Calculator::add(2, 3); // 用 :: 调用
    println!("{}", res); // 5
}
```

## 5\. 关键注意事项

1. **函数顺序无关** ：Rust 不要求函数先定义后使用（编译器会自动查找）
2. **类型严格** ：参数和返回值类型必须精确匹配，不允许隐式类型转换
3. **语句 vs 表达式** ：
	- 带分号： **语句** ，不返回值
		- 不带分号： **表达式** ，返回值（用于函数返回）

---

### 总结

1. 用 `fn` 定义函数，参数必须标注类型
2. 返回值用 `-> 类型` 声明，推荐用 **末尾表达式** 返回
3. 无返回值默认返回 `()`
4. 支持高阶函数、方法等高级用法，是 Rust 编程的基础核心
