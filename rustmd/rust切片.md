## Rust 切片（Slice）

切片是 Rust 中 **最常用的数据类型之一** ，核心作用是 **引用集合中一段连续的元素序列，而不拥有数据所有权** ，非常轻量、高效。

## 一、核心概念

1. **定义** ：切片是对 **数组 / 向量（Vec）等连续内存集合** 的部分引用，类型写作 `&[T]`
2. **特点** ：
	- 没有所有权，只是借用数据
		- 包含 **指针** （指向起始位置）+ **长度**
		- 不可变切片 `&[T]` / 可变切片 `&mut [T]`
3. **适用场景** ：字符串、数组、Vec 等连续存储的数据

---

## 二、基础用法

### 1\. 数组 / 向量切片

```rust
fn main() {
    // 1. 普通数组
    let arr = [1, 2, 3, 4, 5];
    // 切片语法：&集合[起始索引..结束索引]
    let slice = &arr[1..4]; // 取索引 1、2、3 的元素 [2,3,4]

    println!("切片长度：{}", slice.len()); // 3
    println!("切片第一个元素：{}", slice[0]); // 2

    // 2. 动态数组 Vec
    let vec = vec![10, 20, 30, 40];
    let slice2 = &vec[0..2]; // [10,20]
}
```

### 2\. 切片语法简写

Rust 提供了简化写法，非常常用：

```rust
let arr = [1,2,3,4,5];

&arr[..];   // 全部元素（完整切片）
&arr[..3];  // 0~2 索引
&arr[2..];  // 2 到末尾
```

---

## 三、字符串切片（&str）

**字符串切片是 Rust 最重要的切片** ， `&str` 就是字符串切片，也是字符串字面量的类型：

```rust
fn main() {
    let s = String::from("hello rust");

    // 字符串切片
    let slice = &s[0..5]; // "hello"

    // 字符串字面量本身就是切片
    let str_literal = "hello"; // 类型：&str
}
```

> 重点： `String` 拥有所有权， `&str` 是切片（借用），函数参数优先用 `&str` 更通用。

---

## 四、可变切片

如果需要修改切片中的数据，使用 **可变切片** `&mut [T]` ：

```rust
fn main() {
    let mut arr = [1, 2, 3, 4];
    let slice = &mut arr[1..3]; // 可变切片

    slice[0] = 99; // 修改原数组数据

    println!("{:?}", arr); // [1, 99, 3, 4]
}
```

---

## 五、切片作为函数参数（最佳实践）

**这是切片最大的用途** ：函数接收 `&[T]` ，可以同时适配 **数组、Vec、切片** ，通用性极强。

```rust
// 接收切片，兼容数组、Vec、切片本身
fn print_slice(slice: &[i32]) {
    println!("{:?}", slice);
}

fn main() {
    let arr = [1,2,3];
    let vec = vec![4,5,6];
    
    print_slice(&arr);   // 数组传参
    print_slice(&vec);   // Vec 传参
    print_slice(&arr[1..]); // 切片传参
}
```

---

## 六、常用方法

切片自带很多实用方法，无需手动遍历：

```rust
let arr = [10,20,30,40];
let slice = &arr[1..3];

slice.len();       // 长度
slice.is_empty();  // 是否为空
slice.first();     // 第一个元素（Option）
slice.last();      // 最后一个元素
slice.contains(&20); // 是否包含元素
```

---

## 七、关键规则

1. 切片 **不会复制数据** ，仅引用，性能极高
2. 切片索引必须是 `usize` 类型
3. 索引越界会直接 **编译报错 / 运行时 panic**
4. 切片生命周期和原数据绑定，原数据被释放后切片无法使用

---

### 总结

1. **切片 = 指针 + 长度** ，无所有权、轻量高效
2. 语法： `&集合[start..end]` ，支持多种简写
3. 字符串切片 `&str` 是 Rust 字符串的核心类型
4. 函数参数用 `&[T]` ，兼容数组、Vec、切片
5. 可变切片 `&mut [T]` 可修改原数据
