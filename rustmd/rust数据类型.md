## Rust 数据类型（超清晰入门版）

Rust 是 **静态强类型语言** ，编译时就必须确定所有变量类型，不允许类型隐式转换，安全性极高。

数据类型分为两大类：

1. **标量类型** （单个值）
2. **复合类型** （多个值）

---

## 一、标量类型（4 种核心）

代表 **单个值** ，最常用。

### 1\. 整数类型

没有小数的数字，分 **有符号** 和 **无符号** 。

表格

| 类型 | 大小 | 取值范围 | 说明 |
| --- | --- | --- | --- |
| `i8` | 1 字节 | \-128 ~ 127 | 有符号 8 位 |
| `i16` | 2 字节 | \-32768 ~ 32767 | 有符号 16 位 |
| `i32` | 4 字节 | 常用默认 | 有符号 32 位 |
| `i64` | 8 字节 | 大数字 | 有符号 64 位 |
| `u8` | 1 字节 | 0 ~ 255 | 无符号 8 位（字节、颜色） |
| `u16` / `u32` / `u64` | \- | 正数 | 无符号 |

**默认类型** ： `i32`

```rust
let a = 10; // 默认 i32
let b: u8 = 255; // 指定类型
```

### 2\. 浮点类型

带小数的数字。

表格

| 类型 | 大小 | 说明 |
| --- | --- | --- |
| `f32` | 4 字节 | 单精度 |
| `f64` | 8 字节 | **默认类型** ，精度更高 |

```rust
let x = 3.14; // 默认 f64
let y: f32 = 2.0;
```

### 3\. 布尔类型

只有 `true` / `false` ，类型是 `bool` 。

```rust
let is_ok = true;
let finished: bool = false;
```

### 4\. 字符类型

**单个 Unicode 字符** ，用 **单引号** `' '` 。可以是中文、emoji、字母。

```rust
let c = 'z';
let heart = '❤';
let ch = '你';
```

---

## 二、复合类型（把多个值放一起）

代表 **多个值** ，Rust 有 2 种基础复合类型。

### 1\. 数组（Array）

**固定长度 + 相同类型** ，长度不可变。

```rust
// [类型; 长度]
let nums: [i32; 5] = [1, 2, 3, 4, 5];

// 快速创建 [0,0,0]
let zeros = [0; 3];
```

访问元素：

```rust
let first = nums[0]; // 1
```

### 2\. 元组（Tuple）

**固定长度 + 可以不同类型** 。

```rust
// 多种类型混合
let person = ("张三", 18, true);

// 指定类型
let student: (&str, i32, bool) = ("李四", 20, false);
```

访问元素（用 `.索引` ）：

```rust
let name = person.0; // "张三"
let age = person.1; // 18
```

**单元元组** ： `()` ，表示空值。

---

## 三、常用类型速查表

表格

| 分类 | 类型 | 用途 |
| --- | --- | --- |
| 整数 | `i32` `u8` | 数字、计数、字节 |
| 浮点 | `f64` | 小数、数学计算 |
| 布尔 | `bool` | 条件判断 |
| 字符 | `char` | 单个字符 |
| 数组 | `[T; N]` | 固定长度同类型数据 |
| 元组 | `(T1, T2...)` | 临时组合不同类型数据 |

---

### 总结

1. **标量** ： `i32` / `u8` / `f64` / `bool` / `char` （单个值）
2. **复合** ： `数组` （同类型定长）、 `元组` （不同类型定长）
3. Rust 类型安全严格， **不允许隐式类型转换**
