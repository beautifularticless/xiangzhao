## Rust 注释

Rust 提供了 **三种注释方式** ，用途不同，语法简单，我给你整理了最常用、最标准的写法，直接复制就能用。

## 1\. 单行注释（最常用）

使用 `//` ，从 `//` 开始到行尾都是注释

```rust
fn main() {
    // 这是单行注释
    let x = 10; // 也可以写在代码后面
}
```

## 2\. 多行注释

使用 `/* */` ，可以跨多行

```rust
fn main() {
    /*
        这是
        多行
        注释
    */
    let y = 20;
}
```

## 3\. 文档注释（生成官方文档）

Rust 独有的 **文档注释** ，用来生成项目文档，非常重要。

### 对外文档注释（生成 HTML 文档）

`///` ：给 **函数、结构体、模块** 写说明， `cargo doc` 会自动生成文档

```rust
/// 计算两个数的和
/// # 参数
/// * \`a\` - 第一个数字
/// * \`b\` - 第二个数字
/// # 返回值
/// 两个数的和
fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

### 模块 / 包级文档注释

`//!`：用于 **整个文件、模块、crate** 的开头，描述整体功能

```rust
//! 这是一个数学工具库
//! 提供加法、减法等基础运算功能

fn main() {}
```

---

## 快速生成文档

在项目里运行这条命令，Rust 会自动把所有 `///` 和 `//!` 生成漂亮的 HTML 文档：

```bash
cargo doc --open
```

---

### 总结

1. **`//`** ：单行注释，日常写代码用
2. **`/* */`** ：多行注释，临时注释大块代码
3. **`///`** ：函数 / 结构体文档注释（生成文档）
4. **`//!`**：文件 / 模块整体说明（生成文档）
