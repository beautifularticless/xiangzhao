Rust 是一门由 Mozilla 发起、现由 Rust 基金会与社区维护的 **系统级编程语言** ，主打 **内存安全、并发安全、高性能** ，且 **无 GC、无运行时开销** 。 ![image](https://p6-flow-imagex-sign.byteimg.com/labis/image/72539961b3f4328a3e55b71bc868814b~tplv-a9rns2rl98-pc_smart_face_crop-v1:512:384.image?lk3s=8e244e95&rcl=202604242241402AAA3E20EF0C23E91209&rrcfp=cee388b0&x-expires=2092401705&x-signature=Pj42x95HvMV%2B%2FExtKVbb%2F10qJx8%3D)

### 一、核心背景

- 诞生：2006 年由 Graydon Hoare 设计，2015 年发布 1.0 稳定版。
- 定位：替代 C/C++ 的系统级语言，解决其内存安全与并发痛点。
- 口碑：连续多年 Stack Overflow “最受喜爱编程语言”。
- 吉祥物：小螃蟹 Ferris 🦀。

### 二、核心特性（区别于 C/C++/Go）

1. **所有权（Ownership）**
	- 每个值 **唯一所有者** ；所有者离开作用域，内存 **自动释放** ，无需手动 free，也无 GC。
2. **借用与生命周期（Borrowing & Lifetimes）**
	- 可 **不可变借用多个** ，或 **可变借用一个** ，编译期杜绝 **数据竞争、野指针、悬垂引用** 。
3. **零成本抽象**
	- 高级特性（泛型、Trait、闭包）编译为原生代码， **无运行时开销** ，性能比肩 C/C++。
4. **并发安全**
	- 原生支持轻量级线程、通道、锁，所有权模型 **编译期阻止数据竞争** 。
5. **现代工具链**
	- **Cargo** ：一键构建、依赖管理、测试、文档生成。
		- **rustfmt/clippy** ：自动格式化、代码检查。

### 三、适用场景

- **系统编程** ：操作系统（如 Redox）、内核模块、驱动。
- **高性能服务** ：Web 后端（Axum、Rocket）、网络代理、分布式存储。
- **嵌入式开发** ：单片机、IoT、资源受限设备。
- **WebAssembly** ：浏览器高性能应用、游戏、音视频处理。
- **区块链** ：Solana、Polkadot 等项目大量采用。
- **云原生** ：容器、微服务、高性能中间件。

### 四、简单示例（Hello World）

```rust
fn main() {
    println!("Hello, Rust!");
}
```

运行：

```bash
# 安装工具链（Linux/macOS）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# 创建项目
cargo new hello
cd hello
# 运行
cargo run
```

### 五、优缺点

- 优点： **内存安全、并发安全、高性能、无 GC、工具链完善、社区活跃** 。
- 缺点： **学习曲线陡峭** （所有权 / 生命周期难理解）、 **编译时间较长** 、部分领域生态不如 C/C++ 成熟。
